Welcome

We are a group of mobile and web developers learning together to create new and better projects. Our largest project is currently Paste It, an app for uploading text to Stikked. We welcome all contributions to any works.

Some of our services

Redmine

This platform hosts all our projects, and also services as a wiki for all sorts of information.

Gerrit

This code review system originally developed for Android allows anyone to easily push code changes to open source git projects.

Jenkins

This buildbox allows you to compile code on another machine then yours. Ideal for creating ROMs and such.

Stikked

An online platform to share code snippets, code, etc. Simple, secure and fast.

Paste It

Companion app for our Stikked service. Stable Android app, iOS is in the works.

Other stuff

All our apps are open source and hosted on GitHub. We are constantly working on new things.
